# MoonGoons Take Back Web Shell

This folder contains the custom HTML shell used by the **Web Playable** export preset.

The deployed playable build is intended to be served at:

```text
https://potd725.github.io/MoonGoons-Take-Back/
```

The shell gives the Godot export a branded loading frame, touch-friendly viewport settings, and quick control reminders for desktop and mobile testing.
