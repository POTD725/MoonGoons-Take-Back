# Play MoonGoons Take Back Online

The current browser build launches the MoonGoons precinct vertical slice with room repairs, officer selection, distress calls, tactical patrol combat, arrests, research, prisoners, and browser-local save data.

https://potd725.github.io/MoonGoons-Take-Back/

## Deployment status

GitHub Pages is configured to publish through GitHub Actions. A fresh verified deployment was triggered on July 15, 2026 after the Pages source was enabled.

## Browser notes

- Use a modern Chromium, Edge, Firefox, or Safari browser with WebAssembly enabled.
- The game uses a 1280 × 720 landscape viewport and supports mouse and touch input.
- Save data is local to the browser and device.
- On a phone or tablet, rotate to landscape for the intended layout.
- If an older build appears after a new deployment, reload the page without cache.
