# Station Command and Side Ops Release

This release deploys the validated MoonGoons Take Back station-command progression update.

Included systems:

- timed station, Chief's Office, room, equipment, and defense upgrades
- progressively longer upgrade durations by level
- one to four construction slots as the station advances
- 24 individually upgradeable room objects
- engine repair, weapons fitting, medical operations, and interrogation side missions
- guilt, stress, cooperation, credibility, vitals, alignment, coolant, stability, and integrity meters
- point-defense turrets, rail-cannon batteries, shield generators, and interceptor craft
- marauder attack waves, hull damage, shield absorption, repair controls, and defense missions
- expanded chapter, daily, patrol, investigation, district, station, Side Ops, and defense objectives

Validated by the 22-stage Godot test suite, browser export and boot validation, and Android APK export.
